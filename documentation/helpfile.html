Structure

All files and folder are organized, structured and commented with care.

Folder

HTML Structure

Navigation

Navigation links begin on line 69. The href="" is linked to the ID of the sections. So navigation item #features links to section id #features.

This is for the smooth scrolling to it. So if you are changing the names keep in mind that your navigation link has the same href as your section ID.

Logo

There are three logo's in this theme:

1. In fixed navigation

2. On Intro Section

3. In Footer section

Drop your .png logo file inside the images folder and change the img src inside the HTML file to the logo path.

Section Names

Each section has a heading and subtitle (except .brief and .extra section, they only have a heading). You can easily change the content 

Changing Background Images

There are 3 background images used, 

Intro section, newsletter section and video section. You can easily change them by changing the file path inside the stylesheet (style_light.css or style_dark.css, that depends on which version you choosed).

Changing Background Video

If you want a own video as background, place a .ogv or .mp4 video in the video folder. Change the 'data-teaser-source' to the path of the video (dont need to place the extention behind it). Remove the video trigger on line 117. ( Better comment it out if you need to change the video to Vimeo or YouTube in the near future) If you have a vimeo or youtube video that you want to show. Place the id of the video in the 'data-video-src' and change the 'data-provider' to 'vimeo' or 'youtube'. If your lost the video play trigger copy this:

Icons

Font Awesome icons are used in this theme. For more information http://fortawesome.github.io/Font-Awesome/

To see al the available icons of Font Awesome: http://fortawesome.github.io/Font-Awesome/icons/

To change a icon just edit the class names

Example:

change <i class="fa fa-cube"></i> to <i class="fa fa-car"></i>


Animations

Animate.css is used for CSS3 animation. To trigger the animation WOW.js is used. On the website of animate.css you can see all the available animations.

To change an animation edit the class name

Example

change <div class="wow fadeInRight animated"></div> to <div class="wow bounceIn animated"></div>


Features

A single feature item has an icon, h4 and some content p

The middle column of the whole feature section has an image. To change the image edit the file path

Reviews

A single testemonial has an client-image, client-name and client-quote.

Services

A single service item has an icon, h4 and some content p

Newsletter

The subscribe form is only plain HTML, you need to set up/connect a PHP file who handle the connection to the server.

Screenshots

To change the pictures inside the screenshots carousel, just change the img src path

Brief

Brief section has 2 columns, left content and right an image slider. To change the images just edit the img src path, just like above step.

Packages

A single package item has an title h4, price span and a unsorted list of features. At the bottom an order button

Extra Brief

The extra section is a 3 fullscreen tab section.

Each tab is commented

Below the tab content is the tab navigation in a list. Each li contains a button that triggers a tap content. With the ID inside the href

Video

Just replace the iframe code with your own embed iframe code from Vimeo or YouTube.

Download

The download section has 3 buttons. The icons are Font Awesome icons. Just edit the a href to your link destenation.

Tip: Maby you want that links are opened in a new window, place target="_blank" after the href=""

Footer

Footer has 4 columns, and to edit the image, text or links is all link above steps.
